# Social Support and Common Mental Disorders among Parenting Adolescent Girls and Young Women

## Project Overview

This repository contains the Stata code used to investigate the relationship between perceived social support and common mental disorders (CMDs) among parenting adolescent girls and young women (AGYW) enrolled in an ongoing study in south-western Kenya.

The project explored whether social support influences depression, anxiety, post-traumatic stress disorder (PTSD), and the severity of common mental disorders among parenting AGYW.

## Methods

1 Cross-sectional analysis nested within an ongoing  HPV vaccine trial
2 Descriptive statistics
3 Modified Poisson regression
