## Data Availability

The dataset used in this project is **not publicly available**.

The analysis was conducted using data from  an ongoing HPV vaccine clinical trial in Kenya. As the parent study is still in progress, participant-level data are confidential and are not available for public distribution.

This repository therefore **does not include any raw or derived datasets**.

## Reproducibility

The repository provides the Stata scripts used for data management, statistical analysis, and generation of study outputs. These scripts are intended to demonstrate the analytical workflow and statistical methods used in the project.

Users with authorized access to the Study dataset may adapt the scripts to reproduce the analyses.
